# EUSD Claude Code Marketplace

A collection of Claude Code plugins maintained by the EUSD Server Engineering team.

## Installation

Add this marketplace to your Claude Code installation:

```
/plugin marketplace add eusd-server-engineer/claude-code-marketplace
```

## Available Plugins

| Plugin | Description | Category |
|--------|-------------|----------|
| [conversation-extractor](https://github.com/eusd-server-engineer/claude-conversation-extractor-plugin) | Extract Claude Code conversations to markdown files | Productivity |

## Installing a Plugin

After adding the marketplace, install any plugin:

```
/plugin install conversation-extractor
```

## Updating Plugins

To get the latest versions of all plugins:

```
/plugin marketplace update
```

## Contributing

EUSD staff can contribute plugins by:

1. Creating a plugin repository following [Claude Code plugin guidelines](https://docs.anthropic.com/en/docs/claude-code/plugins)
2. Submitting a PR to add the plugin to `marketplace.json`

## Support

For issues or questions, contact the Server Engineering team or open an issue on the relevant plugin repository.
